NLP Sentiment Analysis

A deep learning-based sentiment analysis project using LSTM and GRU models to classify text as Positive, Negative, or Neutral.

-- Features
Text preprocessing using NLTK
LSTM & GRU models
Model training and prediction
Saved models for reuse

--Tech Stack

Python, TensorFlow/Keras, NLTK, Pandas, NumPy

-- Run
python src/train.py
python src/prediction.py
